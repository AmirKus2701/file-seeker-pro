# Автоматическая сортировка и копирование фотографий по папкам коллекций

Автоматическая сортировка и копирование фотографий по папкам коллекций — это утилита на Python, позволяющая обработать большое количество папок с фотографиями и автоматически отобрать из каждой папки нужное фото, а затем рассортировать их по отдельным коллекциям.

## Что делает программа

- Заходит в исходную папку с большим числом вложенных папок (в каждой из них — несколько фотографий).
- Из каждой вложенной папки выбирает одну фотографию, удовлетворяющую условию (например, содержит в имени "_1.jpg").
- Копирует выбранные фотографии в новую папку.
- Копирует фото по коллекциям, определяемым по части имени файла (например, символы 4-5 в имени файла указывают на номер коллекции).
- Если папка коллекции отсутствует, создаёт её.
- Обрабатывает исключения, игнорируя файлы, которые не являются папками.

## Для кого этот проект

- Для фотографов, дизайнеров, менеджеров по контенту и всех, кто работает с большими архивами фотографий.
- Для тех, кто хочет автоматизировать рутинную работу по выбору и сортировке файлов.
- Для начинающих Python-разработчиков, интересующихся практическими задачами работы с файлами и папками.

## Технологии и библиотеки

- Python 3.x
- Модуль os для работы с файловой системой
- Модуль shutil для копирования файлов

## Как пользоваться

1. Поместить программу в папку с исходными каталогами фотографий.
2. Настроить путь исходной папки и папки для копирования в коде.
3. Запустить скрипт.
4. Получить рассортированные фотографии по коллекциям в новой папке.

## Преимущества

- Экономит часы времени за счёт автоматизации монотонной работы.
